# Keyboard Tricks

## For Shell

```bash
# move to the beginnning
CTRL+A
# move to the end
CTRL+E

# move one word forward
ALT+F
# move one word backword
ALT+B

# delete current char
CTRL+D

# kill from current to end of line
CTRL+K
# kill all before current char
CTRL+U
# paste what killed
CTRL+Y
```
